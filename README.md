# Sauti-Apex

Starter repository for Sauti Apex.

## Setup

```bash
npm install
npm run dev
```

## Deployment

This is a TanStack Start server app deployed on Vercel, not a static-only site.
If a custom domain shows a LiteSpeed `Index of /` page, the domain is not being
served by the Vercel deployment. See `DEPLOYMENT.md`.
